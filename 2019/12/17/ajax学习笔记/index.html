ajax学习笔记

发表于 2019-12-17 更新于 2022-10-22 分类于 编程

Asynchronous Javascript And XML (异步的javascript 和 XML)
对页面数据进行局部刷新
创建XMLHttpRequest对象
发送Ajax请求
服务器处理完成后返回响应，通过javascript对响应进行处理

jquery的$.ajax()方法使用：
1、传入的是一个json对象，用于描述发送的细节$.ajax(json);
2、常用的json对象设置：
url 发送请求地址；
type请求类型get/post;
data向服务器传递的参数
dataType服务器响应的数据类型text/json/xml/html/jsonp/script
success 接收响应时的处理函数
error 请求失败时的处理函数
